# 🌸 Iris Flower Classification Using KNN

A machine learning classification project developed as part of the DecodeLabs Artificial Intelligence Industrial Training Program.

## 📌 Project Overview

This project uses supervised machine learning to classify Iris flowers into three species based on their physical measurements.

### Flower Classes

- Setosa
- Versicolor
- Virginica

## 🧠 Machine Learning Pipeline

1. Load the Iris dataset
2. Explore features and target classes
3. Split data into training and testing sets
4. Apply feature scaling using StandardScaler
5. Train a K-Nearest Neighbors classifier
6. Predict flower species
7. Evaluate the model using Accuracy, F1 Score and Confusion Matrix
8. Test the model using custom user input

## ⚙️ Technologies Used

- Python
- Scikit-learn
- Matplotlib
- K-Nearest Neighbors (KNN)
- StandardScaler

## 📊 Model Performance

- Accuracy: 100% on the selected test split
- Weighted F1 Score: 1.00
